> [!NOTE]
>
> ### ClearTemp - приложение которое облегчает жизнь на 5-10%
>Папка Temp по **AppData\Local** известна всем.
>Но когда вы её чистили?
> Я думаю давно.
>
>Но это программа делает всё за вас
>
> Он очищает Temp после...
>
>  **Перезагрузки ПК** **Включение ПК** **После спящего режима**
>
>  *(Если вы его не отключите в автозагрузках Windows)*
>
>  **И его можно запустить вручную!**
>
> И выдаст сообщение сколько очищено ГБ

> [!WARNING]
>
> ### БАГИ
> Мной найден баг что приложение неправильно получает путь к Temp из-за
> **Micrsoft-account**
> И он удаляет неизвестно где. Постараюсь в скором времени это пофиксить

> [!TIP]
> Если вы не знаете где находиться автозагрузка приложения
>
> Она находиться в regedit по пути:
> **Компьютер\HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run**

# ⚙️ Установка и запуск
1. Скачайте ClearTempSetup.exe
 * Он находиться в Release. Под тегом **PreRelease**

2. Запустите его.
  * Установку проводит Inno Setup. Проблем при установке не будет

3. После установки вы можете запустить ClearTemp (ярлык)
  * Если поставить галочку "Запускать ClearTemp при входе в Windows" то будет автозагрузка

# 🔠Описание файлов
 - [**`Temp.exe `**](./Temp.exe) - Основной файл при котором: Чистится Temp, выходят уведомления.
 - [**`InfoStatic.bat `**](./InfoStatic.bat) - вспомогательный файл для уведомлений
 - [**`Unistall.bat `**](./Unistall.bat) - Инсталлятор полностью удаляющий папку ClearTemp и удаляющй из автозагрузки запись про ClearTemp
